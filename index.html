HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The index contains the webpage code.

CSS

  • CSS is a language used to control the appearance of a webpage.
  • CSS stands for cascading style sheets.
  • It describes how elements should be rendered on screen.

Git

  • Git is a distributed version control system.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • All the changes are local only changes.
  • It is used to tracking changes in the source code.
  • It enables multiple developers to work together on a development.

JavaScript

  • In JavaScript you can debug and format.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • It allows to make webpages interactive.
  • Gives webpages interactive elements that engage a use.
  • Is the programming language of the web.
  • It can update and change both HTML and CSS.
  • JavaScript can calculate, manipulate and validate data.
